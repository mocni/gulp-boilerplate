Gulp Boilerplate
=============
 
## Install
```js
npm install gulp-boilerplate -g
```

## CLI
Once you've installed gulp-boilerplate you can then run the command `gulp-boilerplate` to add gulp to your project.

```js
gulp-boilerplate
```
